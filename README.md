# Kurdish Word sorter
A tool for sorting kurdish word document by accepting word xml

**Packages**
* bundler: Parcel
* CSS Library: Milligram css
* XML parser: node-html-parser

## Development
inorder to run project in development mode run: 
```bash
yarn install

yarn dev
```

## Build
to build project run following commands:
```bash
yarn install

yarn build
```
